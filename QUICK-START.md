## ğŸ†˜ Quick Fixes

### Hand Not Detected?
1. Check lighting (turn on more lights)
2. Move closer/farther from camera
3. Use plain background
4. Refresh browser (Ctrl+R)

### Gestures Not Working?
1. Hold gesture longer (1 second)
2. Make bigger, clearer movements
3. Check camera can see full hand
4. Try the other hand

### App Won't Open?
1. Make sure you're pinching (thumb + index)
2. Hold pinch for full second
3. Point directly at the icon
4. Refresh page if stuck
